Poetry Web Splicer README


This is a model view controller web application built with .NET. The programmer I am working with primarily uses C#. We are doing this in stages to ensure all pieces work together properly.

start page skeleton:
    Homepage. Pressing "generate" generates a phrase from pre-programmed text files (temporary for testing).

generated phrase and generated phrase 2:
    Samples returned after hitting "generate".

data:
    Tab where entries are pulled from the database. 

content detail:
    Detail of the entry. It is possible to edit or delete the entry.

create new:
    New data is added to database.
